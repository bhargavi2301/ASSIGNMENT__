10.Which property controls the image scroll in the background?

The "background-attachment" property can take one of the following values:
scroll: This is the default value. It allows the background image to scroll along with the content as the user scrolls down the page.
fixed: value makes the background image stay fixed in place relative to the viewport. As the user scrolls, the background image does not move, creating a "fixed" background effect.

11.Why should background and color be used as separate properties?

Accessibility: separating background and text color, you can ensure that your content is accessible to a wide range of users, including those with visual impairments. This is a fundamental principle of web accessibility.
Design Flexibility: background and text color allows for greater design flexibility. You can easily change the background color or image without affecting the text color, and vice versa. This makes it easier to experiment with different design choices.
Readability: Combining background and text color can make the content less readable. For example, using a light text color on a light background or a dark text color on a dark background can strain users' eyes and decrease readability.
Responsive Design: In responsive web design, the layout and design of a website may change based on the user's device and screen size. Separating background and text color helps maintain consistency and adaptability in different layouts.
Print Styles: When preparing a website for print, having separate properties for text and background colors allows you to create styles optimized for print, which often differ from web styles.
Consistency: these properties helps maintain consistency across your website. You can define a consistent text color for all elements, making it easier to manage and achieve a cohesive design.
Maintenance: When background and text color are separate, it's easier to maintain and update your website. If you decide to change the background color or text color, you only need to modify one property, rather than revisiting all elements that combine both properties.

12.How to center block elements using CSS1?

In CSS1, there was no direct method for centering block elements both horizontally and vertically, as modern CSS provides. However, you could use some HTML and CSS tricks to achieve basic centering. Note that CSS1 is very limited compared to modern CSS, so these methods may not be as flexible or reliable as what's available in later versions of CSS.
Centering Horizontally:To center a block element horizontally within its containing element, you can set its left and right margins to "auto."
Centering Vertically:In CSS1, vertically centering block elements was more challenging and often required you to know the height of the container and the centered element

14. What is embedded style sheets?

Embedded stylesheets, also known as embedded CSS,are a way to define and apply styling rules directly within
an HTML document. CSS is used for controlling the visual presentation of web pages, such as the layout, colors, fonts, and other design elements.
There are three main methods for including CSS in an HTML document: inline styles, embedded styles, and external styles.

15. What are the external style sheets?

An external style sheet is a separate CSS file that can be accessed by creating a link within the head section of the webpage.

16.What are the advantages and disadvantages of using external style sheets?

1. Consistency: External style sheets allow you to maintain a consistent look and feel across your entire website.
By defining styles in one central location, you can ensure that all your web pages have a uniform appearance.

2. Ease of Maintenance: Making changes to your website's design becomes easier with external style sheets.
You can update the style sheet once, and the changes will automatically apply to all pages that link to it. This saves time and reduces the risk of errors.

3.Faster Loading Times: External style sheets are cached by the browser after the initial download.
This means that subsequent pages will load faster because they don't need to download the same styles again. This can improve the overall performance of your website.

Disadvantages of External CSS: Your pages may not be rendered correctly until the external CSS is loaded.
Uploading or linking to multiple CSS files can increase your site's download time.

17. What is the meaning of the CSS selector?

In CSS, a selector is a pattern that is used to select and target specific HTML elements on a web page.
CSS selectors are used to define the style rules that should be applied to those selected elements.
They play a crucial role in the separation of content from presentation on a webpage. Selectors determine which elements the
CSS rules should be applied to.
