8.What is the use of the background-position property?

Controlling the repetition of a background image in CSS can be done using the background-repeat property.
This property allows you to specify whether and how the background image should repeat in both the horizontal and vertical directions.
There are several values you can use for background-repeat:
repeat:This is the default value. It repeats the background image both horizontally and vertically.
the image is smaller than the element it's applied to, it will repeat to fill the entire element.

1.repeat-x: This value repeats the background image only horizontally, creating a horizontal stripe effect.

2.repeat-y: This value repeats the background image only vertically, creating a vertical stripe effect.

3.no-repeat: This value prevents the background image from repeating in either direction, so it will only appear once.

4.space: This value repeats the background image both horizontally and vertically, but it adjusts the spacing between the repeated images so that they fill the container without cropping.

5.round: Similar to space, this value repeats the background image both horizontally and vertically, but it also scales the image up or down to ensure it fits the container without cropping.

10.Which property controls the image scroll in the background?

The "background-attachment" property can take one of the following values:
scroll: This is the default value. It allows the background image to scroll along with the content as the user scrolls down the page.
fixed: value makes the background image stay fixed in place relative to the viewport. As the user scrolls, the background image does not move, creating a "fixed" background effect.
