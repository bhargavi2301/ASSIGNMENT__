Q.1) What are the new tags added in HTML5?

Ans.These new elements add advanced functional features, improvised page layouts, and other improvements.The commonly used new HTML5 tags are figure, article, header, main,footer, nav, and section.

Q.2) How to embed audio and video in a webpage?

Ans.The audio element is used to embed audio files to a web page, and the video element is used to embed a video.

Q.3) Semantic element in HTML5?

Ans. A semantic element clearly describes its meaning to both the browser and the developer. Examples of non-semantic elements:
and - Tells nothing about its content. Examples of semantic elements: form , table , and article - Clearly defines its content.

Q.4)Canvas and SVG tags

Ans.SVG is a language for describing 2D graphics in XML.
Canvas draws 2D graphics, on the fly (with JavaScript).
SVG is XML based, which means that every element is available within the SVG DOM. You can attach JavaScript event handlers for an element.
In SVG, each drawn shape is remembered as an object.
